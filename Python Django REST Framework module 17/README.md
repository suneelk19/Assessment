# EduTracker REST API

## Features

- Student CRUD
- Course CRUD
- Many-to-Many Relationship
- Search
- Filter
- Ordering
- Pagination
- Django Admin
- REST API

## Technology

- Python
- Django
- Django REST Framework
- SQLite

## Run Project

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
