# WriteSphere

## Features
- User Registration
- Login / Logout
- Create Post
- Edit Post
- Delete Post
- Image Upload
- Categories
- Tags
- Search
- Category Filter
- Author Filter
- Date Filter
- Comments
- Like / Unlike
- Follow / Unfollow
- Pagination

## Technologies
- Python
- Django
- MySQL
- HTML
- CSS
- Bootstrap

## Installation

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
